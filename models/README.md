### 1. 智谱 AI GLM-4-Flash ⭐⭐⭐⭐⭐

**特点：**
- ✅ **最便宜**: ¥0.1/1M tokens (约 $0.014)
- ✅ **速度快**: 响应迅速
- ✅ **中文优化**: 对中文理解好

**适合场景：**

- 文档总结

- 数据分析

**注册流程：**
1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
2. 注册账号 (支持手机号/微信)
3. 实名认证 (必须，否则无法充值)
4. 充值 (最低 10 元)
5. 创建 API Key

**配置示例：**
```json
{
  "model": "zhipu",
  "apiKey": "你的 API_KEY",
  "modelId": "glm-4-flash"
}
```

## 🔑 API Key 管理

### 安全建议
1. **不要泄露**: 不要提交到 GitHub
2. **定期轮换**: 每 3 个月更换一次
3. **权限最小**: 只开通需要的权限
4. **监控用量**: 设置用量告警

### 环境变量配置
```bash
# ~/.bashrc 或 ~/.zshrc
export ZHIPU_API_KEY="你的密钥"
export BAILIAN_API_KEY="你的密钥"
```

### OpenClaw 配置
```json
// ~/.openclaw/config.json
{
  "models": {
    "zhipu": {
      "apiKey": "你的密钥",
      "modelId": "glm-4-flash"
    }
  }
}
```

## 🆘 常见问题

### Q: 实名认证失败？
- 检查身份证照片是否清晰
- 确保姓名和身份证号一致
- 联系客服处理

### Q: 充值后余额不显示？
- 刷新页面
- 退出重登
- 联系客服查询

### Q: API 调用失败？
- 检查 API Key 是否正确
- 确认账户有余额
- 查看错误信息
